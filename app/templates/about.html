About F1 Racing App

Your comprehensive Formula 1 team and driver management system

What is F1 Racing App?

This web application is designed to manage and display Formula 1 racing data for the 2025 season. Built with Flask and MySQL, it provides a comprehensive platform for tracking all 10 F1 teams and 20 drivers with full CRUD (Create, Read, Update, Delete) functionality.

Database Management

Utilizes AWS RDS MySQL database with proper foreign key relationships and CASCADE operations to maintain data integrity. Team and driver information is stored with full relational support.

CRUD Operations

Perform complete Create, Read, Update, and Delete operations on teams and drivers. Add new teams, update driver rosters, assign drivers to teams, and manage all racing data.

Team Branding

Each team features authentic branding including team colors, logos, and car images. The interface dynamically displays team information with visual elements for better user experience.

Driver Profiles

Comprehensive driver profiles with photos, nationalities, driver numbers, dates of birth, and team assignments. All 20 drivers for the 2025 season are included with official imagery.

Technologies Used

  • Flask: Python web framework for backend
  • MySQL: AWS RDS hosted relational database
  • Bootstrap 5: Responsive CSS framework
  • Jinja2: Template engine for dynamic content
  • Font Awesome: Icon library
  • PyMySQL: MySQL database connector
